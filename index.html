Projects

Flight Simulator Suite

FlightGear is an open source cross platform flight simulator. It has been in development for many years and has accumulated many rich features. Some of these features include being able to fly many aircraft, exploring locations around the globe, and experiencing a hyper realistic simulation of aviation. The goal of this project was to build upon FlightGear. We added functionalities including a new wxPython GUI and a message queuing subsystem within FlightGear to communicate with the GUI. The primary challenge of this project was safely adding custom components to a complex piece of existing software. Each part of the project involved a deep understanding of the relevant portions of code. The new GUI was built using wxPython. The reason behind choosing Python is simply because Python is such an easy language to understand and write, thereby making it easier for future developers of this project to build upon what we have created with more ease. The wxPython GUI library was chosen because it is a community favorite and has much support on the web. The message queuing subsystems within FlightGear were implemented using ActiveMQ. ActiveMQ uses a broker to mediate the communication between multiple processes. It also has libraries written in C++ and Python which we used to connect FlightGear and the GUI to the broker. My role in this project was implementing the message producer and consumer subsystems within FlightGear and making sure that FlightGear and the Python GUI would be able to connect to the broker.

Space Game

This game involves flying in a spaceship through an asteriod field. Your score increases the longer you survive. To survive, avoid hitting the ceiling, floor, and all the asteroids that come your way. Occasionally a power-up(a green square) will spawn. Flying over a power-up will increase your ship's speed while making you invunerable to asteroids. Your speed is increased for the duration but the invunerablility lasts for an additional two seconds. The power-ups do not make you invunerable to the floor and ceiling however. Aquiring the power-up also increases your score multiplier for the duration of the power-up. The game will also keep track of your high score. This game was developed using C#, DirectX, and Visual Studio. View this project on GitHub.
